## 주제

해당 프로젝트에서는 회원 등록, 입금, 출금, 회원 목록, 회원 탈퇴, 회원 현재 저금액, 프로그램 종료까지 구현합니다. 

프로그램 종료를 한 뒤 다시 프로그램을 진행했을 시, 이전 기록이 남아있기 위해선 데이터베이스까지 동원하여 이전 기록을 다시 떠오르게끔 구현해야겠지만 일단 프로그램으로 기본적인 틀을 만드는 것이 이번 프로젝트의 목표입니다.

```text
3. while True, break, continue로 반복 흐름을 제어합니다.

5. q 문자열을 누를 시에 프로그램이 종료되는 기능을 만듭니다.
6. 저장된 값을 딕셔너리에 넣어 원하는 자료가 출력되는 기능을 만듭니다.
```

## 회원 등록

### 계정명

```text
이용자가 회원등록을 할 때 자신의 회원명과 첫 입금할 금액을 입력합니다.
영어로만 기입합니다. 예시: oh byung hoon
계정이 생성된 후 계정명과 현재 저금액을 메세지로 띄웁니다.
계정명은 영어와 숫자로만 연속적으로 생성이 되게끔 기능을 구현합니다.
```

## 입금

### 입금액

```text
이용자가 자신의 계좌에 금액을 입금할 때 먼저 회원명과 입금액을 기입합니다.
자신이 계정을 생성했을 때 회원명을 기입합니다.
입금한 후 자신의 계정에 얼마나 저축되어있는지 금액을 확인하게끔 메세지를 띄웁니다.
```

## 출금

### 출금액

```text
이용자가 자신의 계좌에서 출금을 원할 때 먼저 회원명과 출금액을 기입합니다.
자신의 회원명을 기입합니다.
출금한 후 자신의 계정에 얼마나 저축되어있는지 금액을 확인하게끔 메세지를 띄웁니다.
```

## 회원 목록

### 회원 목록 조회하기

```text
이용자가 전체 이용자 목록을 조회하는 건 프로그램 상 이치에 맞지 않지만 어디까지나 연습용 프로젝트이기 때문에 해당 프로그램에서 기능을 구현해봅니다.
회원 목록 조회하기 메뉴로 가서 현재까지 누적된 회원들과 회원들이 누적했던 금액들을 조회합니다.
```

## 회원 탈퇴

### 회원 탈퇴하기

```text
탈퇴를 원하는 이용자가 있을 시 자신의 저금액을 전부 출금하고 계정을 탈퇴합니다.
먼저 출금하기 메뉴로 가서 자신의 저금액을 전부 출금해서 0원으로 만든 후에 계정 탈퇴가 가능하다는 메세지를 띄웁니다.
처음 메뉴로 돌아가서 출금하기 메뉴로 들어가 0원으로 만든 후에 회원 탈퇴하기 메뉴로 들어가서 회원 탈퇴를 진행합니다.
```

## 회원 저금액

### 회원 현재 금액

```text
현재 자신이 저축한 금액이 얼마나 되는지 확인하는 메뉴를 만듭니다.
해당 메뉴로 들어가서 자신의 회원명을 기입한 뒤 저축된 금액이 얼마인지 확인합니다.
```

## 프로그램 종료

### 프로그램 종료

```text
프로그램을 시작하고 난 후 메뉴 중 프로그램 종료를 할 수 있는 메뉴를 띄웁니다.
백엔드 시스템 상 데이터 베이스에 모든 기록들을 저장해서 이전 기록을 불러오는 기능을 구현해야겠지만 취지가 미니 프로젝트로서 프로그램의 틀을 만드는 것이 목표이기 때문에 이전 기록들은 저장되지 않고 프로그램만 off되도록 기능을 만듭니다.
```
